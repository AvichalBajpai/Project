# 🏏 Cricket Game (Bat Ball Stump)

A simple interactive cricket game built using HTML, CSS, and JavaScript.

## 🚀 Live Demo
👉 https://avichalbajpai.github.io/Cricket-Game/

## 🎮 How to Play
- Choose one: Bat 🏏, Ball ⚾, or Stump 🪵
- Computer will randomly choose
- Winner is decided based on rules:
  - Bat beats Ball
  - Ball beats Stump
  - Stump beats Bat

## ✨ Features
- User vs Computer gameplay
- Random computer choice
- Live result display
- Score tracking (Win / Loss / Draw)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
